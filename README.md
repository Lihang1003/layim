# webim介绍
基于layim和weui开发，支持音视频通话！

#### 功能
- 基础通讯
- 聊天记录
- 添加好友
- 添加群组
- 创建群组
- 好友私聊
- 好友群聊
- 离线消息
- 发送文件
- 上下线通知
- 好友分组管理
- 查找好友
- 消息撤回
- 删除好友
- 退出群组
- 好友在线状态（实时准确）
- 异常掉线处理
- 修改个人信息
- 接收消息
- 好友移动分组
- 查看好友详细资料
- 修改备注
- 消息加解密
- 群聊管理（踢人，拉人，禁言，管理员设置等）
- 群聊艾特
- webrtc音视频通话(没有配置turn服务器只能在同一个局域网下使用该功能) :exclamation:  :exclamation:  :exclamation: 
- 发红包 :exclamation: 
- 扫码添加好友 :exclamation: 
- pc端扫码登录
- 拍一拍
- 朋友圈等...
